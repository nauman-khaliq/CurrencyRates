
# Currency Rates 

Sample Android App for demostrating the use of MVVM architecture with Jetpack Compose ui and Jetpack Compose navigation.


## Tech Stack

- Kotlin v1.9.10
- jvm 17
- Gradle v9.0
- HILT v2.4.8
- Moshi v1.15
- Room v2.5.2
- ksp v(1.9.10-1.0.13)
- Retrofit v2.9.0
- Coil 2.4.0
- Jetpack Navigation Component v2.7.1
- Material 2
- Kotlin Flows
- State Flows
- Jetpack Compose UI
- Jetpack Compose Navigation
